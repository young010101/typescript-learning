# 01 - TypeScript 基础

## 学习目标
- 了解 TypeScript 基本概念
- 掌握基本语法和类型注解
- 学会编译和运行 TypeScript

## 内容列表
1. `hello-world.ts` - Hello World 程序
2. `variables.ts` - 变量声明和类型注解
3. `primitive-types.ts` - 基本数据类型
4. `type-inference.ts` - 类型推断
5. `union-types.ts` - 联合类型

## 关键概念
- 类型注解 (Type Annotations)
- 类型推断 (Type Inference)
- 编译时类型检查
